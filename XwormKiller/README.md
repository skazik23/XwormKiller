

## Description

XwormKiller is a specialized utility designed to detect and eliminate XWorm RAT and similar Remote Access Trojans from infected Windows systems.

## Features

- Real-time process monitoring
- System process integrity check (svchost.exe, explorer.exe, MSBuild.exe)
- Active port scanning and blocking
- Firewall rules for RAT ports
- Registry cleanup (autorun, tasks)
- Cache wipe
- Colored console output
- Self-protection

### Prerequisites
- Windows 10/11 (x64)
- Administrator privileges
